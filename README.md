# 🤹🏻 로또 번호 추첨기 🤹🏻

## 📌 프로젝트 설명
이 프로젝트는 1부터 45까지의 숫자 중 랜덤으로 6개의 로또 번호를 추첨해주는 웹 애플리케이션입니다.

## 🎯 주요 기능

- **1~45 숫자 중 랜덤 6개 추첨**
- **숫자 범위별 컬러 구분**
- **연속 요청 방지**
- **반응형 UI**

## ✨ 화면 예시

> 번호가 추첨될 때, 숫자에 따라 색상이 적용되고 하나씩 순차적으로 나타납니다.


## ⏳ 향후 추가 예정 기능

- **최근 1개월 간의 당첨 번호 가져오기**
- **번호 기록 저장 및 불러오기**
- **번호 통계 기능 (가장 많이 나온 번호 등)**
